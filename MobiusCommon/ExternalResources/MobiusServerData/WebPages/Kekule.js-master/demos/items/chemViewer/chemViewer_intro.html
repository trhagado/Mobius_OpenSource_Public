ChemViewer is the most important widget to display molecules and other chemical objects in 2D or 3D mode. It can be used in stand-alone web application or embedded web page.

To add a viewer widget, the easiest method is using the widget "auto-launch" feature describled in Auto-Launch demo, for example:

<div id="chemViewer2D" style="width:600px;height:400px"
  data-widget="Kekule.ChemWidget.Viewer2D"></div> <!-- a 2D viewer -->
<div id="chemViewer3D" style="width:600px;height:400px"
  data-widget="Kekule.ChemWidget.Viewer3D"></div> <!-- a 3D viewer -->

then refer to them in JavaScript code:

var chemViewer2D = Kekule.Widget.getWidgetById('chemViewer2D');
var chemViewer3D = Kekule.Widget.getWidgetById('chemViewer3D');

Of course, you can also create a new viewer:

var viewer = new Kekule.ChemWidget.Viewer2D(document);
viewer.appendToElem(document.body);

To load an object in viewer:

viewer.load(molecule);

or:

viewer.setChemObj(molecule);

By the way, molecule and other chem object data can also be embedded in HTML, just see this demo.
