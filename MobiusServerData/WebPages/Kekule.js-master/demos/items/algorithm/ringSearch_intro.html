Molecule ring perception is a common task in chemoinformatics system. Kekule.js provides built-in method to detect all rings, SSSR (smallest set of smallest rings) and even aromatic rings. The code is simple, suppose variable molecule is an instance of Kekule.StructureFragment:

var allRings = molecule.findAllRings();
var sssr = molecule.findSSSR();
var aromaticRings = molecule.perceiveAromaticRings();

Each result of three different methods is a ring array. Each element in array is a plain JavaScript object with fields:

nodes: Array of all nodes (atoms) in ring.
connectors: Array of all connectors (bonds) in ring.
